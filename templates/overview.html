User Guide

Welcome to Bug Tracker!

See the full documentation here.

In the overview page, tickets shown in the column chart are organized by type, and the project they are filed in.

Click on the column section to go to the tickets page.

The ticket list is filtered depending on which Ticket type and Project you choose.

Critical priority tickets are shown in the overview page, along with Resolved tickets.

See the table below regarding User permissions.

Role Permissions
Admin
  • Create Projects
  • Submit Tickets
  • Be assigned to tickets
  • Resolve Tickets
  • Delete Users
Project Manager
  • Create Projects
  • Delete Projects
Developer
  • Be assigned to tickets
  • Resolve Tickets
  • Delete Tickets
Submitter
  • Submit new tickets
